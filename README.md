# trashyicons
Another icon pack nobody needed.

Example code:

<code>&lt;i class="ti-exclamation" id="myExampleIcon">&lt;/i></code>

Change icon size:

<code>&lt;script>
  window.addEventListener('load', function() {
    document.getElementById('myExampleIcon').childNodes[0].style.height = "20px";
  })
&lt;/script></code>
